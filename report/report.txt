## Conclusion

Metasploitable 2 exposes **23 open TCP ports** with **10 critically exploitable services**. The vulnerability scan directly confirmed root-level code execution via the vsftpd backdoor (CVE-2011-2523), multiple SQL injection endpoints on port 80, RMI remote classloader RCE on port 1099, and SSL/TLS weaknesses (POODLE, CCS Injection, Logjam) on ports 25 and 5432.

Key takeaways from this exercise:
- Outdated software versions are the primary attack surface
- Backdoored binaries (vsftpd, UnrealIRCd) are a real-world supply chain threat
- Unencrypted protocols (Telnet, rsh, rlogin, FTP) should never be used in production
- Anonymous FTP and guest SMB access dramatically reduce the effort needed to compromise a system
- Web applications need input validation — SQLi and CSRF were confirmed across multiple endpoints
- A single misconfigured service is enough for full system compromise
